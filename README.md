# Project-One
First RStudio Project in MPH716, using Gapminder to compare two countries: Guatemala vs. United States. This project analyzes long‑term trends in life expectancy, GDP per capita, and population for Guatemala, using the United States as a comparison. Using R, tidyverse, and the Gapminder dataset (1952–2007), the project generates descriptive statistics and visualizations to show how economic conditions and structural inequities shape population health. The analysis connects quantitative trends with public health literature on rural care, Indigenous health disparities, and historical context in Guatemala. The project also reflects on data limitations, such as five‑year intervals and national‑level aggregation – and demonstrates how reproducible workflows support transparent, updateable public health storytelling.
